# 主宰面板 (Commander Dashboard)

**目标：** 做一个个人管理面板，可视化追踪项目、目标、周计划、未来规划、思考与行动，释放认知负荷
**状态：** 🎯 规划中
**起始：** 2026-05-25
**负责人：** 主宰

## 要解决的问题

用户（Zeno）需要在脑中追踪所有事务——项目进度、长短期目标、每周安排、未来规划、务虚的思考和务实的行动。这消耗大量脑力。需要**一个可视化的工具**让一切变得可见、轻松、可调控。

## 定位

- 纯个人使用，不考虑多人协作
- 手机 + 电脑双端可用
- 涵盖：生活、工作、思考（务虚）、实践（务实）
- 一页即可看全局

## 核心原则

1. **可视化优先** — 数据要能一眼看懂，信息层级清晰
2. **零认知负荷** — 打开即用，不需要学习操作
3. **本地优先** — localStorage + 导出备份，不上传任何数据
4. **可进化** — 功能从简到繁，不提前设计
