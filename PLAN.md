## 六、数据库结构

### projects 表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | UUID | 主键 |
| name | TEXT | 项目名 |
| status | TEXT | planning/in_progress/completed/archived |
| progress | INTEGER | 0-100 |
| tags | JSONB | 标签数组 |
| description | TEXT | 描述 |
| lead | TEXT | 负责人 |
| docs | JSONB | 文档数组 [{title, content, type}] |
